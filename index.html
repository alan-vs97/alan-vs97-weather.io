Summary: Latitude vs. X

The purpose of this project was to analyse how different aspects of the weather change as you approach the equator from the poles. To accomplish this, we pulled data from the OpenWeather API, choosing a sample of 500 cities at random.


Once we gathered the data, we plotted the results using python's Matplotlib library. The factors we were most interested in were: temperature, cloudiness, wind speed and humidity. This site provides the source data used in the analysis, as well as the plots we produced, and a small analysis of the trends we observed.
